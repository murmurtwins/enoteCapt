【このソフトはどんなもの？】

シャープ株式会社から販売されている電子ノートシリーズを
活用するべく開発したソフトウエアです。

パソコン画面をキャプチャして電子ノートシリーズで読み込める
ビットまマップ形式（１６諧調のグレースケール）として保存できます。

Windows 10 / 64bitにて動作確認しています。

【使い方】

enoteCapt.exe を適当なフォルダに置き実行してください。

起動するとウインドウが開きます。

「キャプチャ」ボタン
　キャプチャボタンをクリックすると画面上のどこをキャプチャするか
　枠が表示されますのでマウスの右ボタンクリック／シフトキーを押すで
　キャプチャが実行され、取り込まれた画像が下に表示されます。

「保存」ボタン
　取り込まれた画像をビットマップファイルとして保存されます
　ビットマップファイルはenoteCapt.exeと同じフォルダに保存されます。

「明るさ」
　画像の明るさを調整できます。

「コントラスト」
　画像のコントラストを調整します。

取り込まれた画像をそのまま保存すると電子ノートでは見ずらい画像になることがあるので
明るさやコントラストを調整してみやすくして保存すると見やすくなります。
生成された画像は電子ノートのIMPORTフォルダにコピーすることでそのまま取り込めます。
